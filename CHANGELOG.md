# Changelog

本文件记录每个公开版本的用户可感知变化，便于后续发布、回滚和问题追溯。

## [0.1.1] - 2026-05-01

### 新增

- 增加版本展示，标题栏会显示当前构建版本号。
- 增加版本管理脚本，支持校验和同步 `desktop/package.json`、`desktop/src-tauri/Cargo.toml`、`desktop/src-tauri/tauri.conf.json` 的版本号。

### 调整

- 首页 GitHub 仓库链接恢复为新页面打开，不再覆盖当前界面。

## [0.1.0] - 2026-04-30

### 新增

- 首个开源版本，包含桌面端与纯 Web 端界面、额度查询、优先级草稿、备份和同步能力。
