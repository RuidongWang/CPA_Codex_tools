# Changelog

本文件记录每个公开版本的用户可感知变化，便于后续发布、回滚和问题追溯。

## [Unreleased]

## [0.1.3] - 2026-05-04

### 新增

- 增加 Keeper 功能，支持在账号列表中指定账号后直接禁用、刷新证书和删除证书。

### 调整

- 完善额度获取流程，完成额度获取后会把账号列表与额度快照持久化到本地缓存。

## [0.1.2] - 2026-05-03

### 调整

- 项目入口收敛为 Web-only，本地使用通过浏览器或 Docker Compose 启动。
- 移除 Python CLI、桌面壳层、桌面构建脚本和相关测试。
- 前端工程目录从 `desktop/` 调整为 `web/`。

## [0.1.1] - 2026-05-01

### 新增

- 增加版本展示，标题栏会显示当前构建版本号。

### 调整

- 首页 GitHub 仓库链接恢复为新页面打开，不再覆盖当前界面。

## [0.1.0] - 2026-04-30

### 新增

- 首个开源版本，包含 Web 界面、额度查询、优先级草稿、下载和同步能力。
